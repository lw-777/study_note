操作系统
{
	功能
	{
		1、进程管理
		2、内存管理
		3、设备管理
		4、文件管理
		5、网络管理
		
		包括进程管理和内存管理功能就是操作系统
	}
	
	一个大型的系统程序
	{
		1、提供用户接口
		2、为应用程序分配和调度软硬件资源
	}
	
	单道批处理系统特点
	{
		1、批量
		2、自动
		3、单道
		4、CPU利用率低
	}
	
	多道批处理系统特点
	{
		1、多道
		2、宏观并行
		3、微观串行
		4、作业处理时间边长
	}
	
	中断技术和通道技术
	{
		通道技术：处理内存和外设间数据传输
	}
	
	分时操作系统:以时间片为单位
		Unix：第一个实用化的分时系统，实现可移植，实现硬件无关
	
	实时系统
	{
		1、硬实时系统
		2、软实时系统
	}
